# Cybersecurity Portfolio
Cybersecurity portfolio showcasing hands-on labs, scripts, and write-ups focused on network analysis, vulnerability enumeration, Linux, and Python. This repository highlights practical experience with tools such as Wireshark, Nmap, Kali Linux, and custom scripting in controlled lab environments.

## Featured Projects

- **Vulnerability Enumeration Lab** — Built an isolated virtual lab using Kali Linux and Metasploitable2 to perform reconnaissance, identify open ports, and examine potential weaknesses with Nmap.
- **Network Traffic Analysis** — Captured and analyzed network traffic in Wireshark to distinguish encrypted and plaintext traffic and identify suspicious patterns.
- **Python Password Auditing Tool** — Developed a Python-based tool using Regex to evaluate password strength and support credential hygiene analysis.

## Tools and Technologies
Wireshark, Nmap, Kali Linux, Metasploitable2, Python, Linux CLI, VirtualBox
